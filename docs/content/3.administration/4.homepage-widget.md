# Homepage Dashboard Widget

This guide provides instructions for adding the SparkyFitness service widget to your **Homepage** dashboard.

**Note: This widget is maintained by the Homepage developer community.**

The SparkyFitness widget displays a summary of your health stats (e.g., active goals, calorie intake) directly on your personal dashboard.

For detailed configuration instructions, API key requirements, and widget options, please refer to the official Homepage documentation:

[Homepage SparkyFitness Widget Documentation](https://gethomepage.dev/widgets/services/sparkyfitness/)
